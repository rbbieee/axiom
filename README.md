# AXIOM

An AI agent product landing page built as a frontend portfolio project.

The concept behind AXIOM is a silent, autonomous intelligence. Not an assistant, but an entity that operates independently and surfaces only when there is something worth knowing. The design reflects that: dark, deliberate, and minimal.

## Tech Stack

- Next.js 
- TypeScript
- Tailwind CSS 

## Running Locally

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
